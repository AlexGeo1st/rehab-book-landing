# rehab-book-landing (v5)

Обновлённая сборка лендинга-презентации книги (учебная практика).

## Что добавлено в v5
- Встроено **видео по спортивному тейпированию** прямо в секцию `#taping` (HTML5 `<video>`)
- Видео хранится локально в репозитории: `assets/video/taping_cska.mp4`
- Добавлен постер к видео: `assets/img/taping_poster.jpg`

## Запуск локально
Открой `index.html` двойным кликом или через Live Server в VS Code.

## Публикация на GitHub Pages
1) Залей файлы в репозиторий (ветка `main`, корень проекта).
2) Settings → Pages → Deploy from a branch → `main` / `(root)` → Save.
3) Открой: https://alexgeo1st.github.io/rehab-book-landing/

> Примечание: GitHub Pages обслуживает статические файлы — для видео это подходит, но первый прогруз может быть медленнее на мобильном интернете.
