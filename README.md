# rehab-book-landing (v3)

Лендинг-презентация книги (учебная практика).

## Что внутри
- Адаптивная вёрстка (Grid/Flex)
- Интерактив (чистый JS): бургер-меню, табы, модалка с формой, слайдер-галерея
- Интерактивный фон: canvas-партиклы (учитывает prefers-reduced-motion)

## Структура
- `index.html`
- `css/style.css`
- `js/main.js`
- `assets/img/*`
