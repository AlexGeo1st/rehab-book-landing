# rehab-book-landing (v7)

Учебный лендинг-презентация книги «Возвращение в игру» (Frontend-разработка).

## Новое в версии v7
- Добавлена отдельная мини-секция **«Анонсы и планы развития книги»**
- Анонсированы обновления книги, запланированные на 2026 год
- Блок размещён перед футером и логически завершает основной контент страницы

## Сохранённый функционал
- бургер-меню
- табы (кейсы травм)
- модальное окно «Оставить заявку» (учебная форма)
- слайдер изображений с миниатюрами
- встроенное видео спортивного тейпирования
- адаптивная вёрстка

## Публикация
Проект развёртывается на GitHub Pages и не требует серверной части.
