# rehab-book-landing (v6)

Сборка лендинга-презентации книги (учебная практика).

## Изменения v6
- Видео в разделе **Тейпирование** отображается в горизонтальном формате (aspect-ratio 16:9)
- Постер видео — скриншот **первого кадра** (taping_poster.jpg)
- Секции переставлены: **Галерея** теперь выше, а **Тейпирование** ниже
- Сохранены интерактивные элементы: бургер-меню, табы, модалка, слайдер + миниатюры
